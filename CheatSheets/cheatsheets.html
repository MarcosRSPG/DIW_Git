Resumen estructurado basado en cheatsheets típicos de HTML5 y CSS3.

Cómo usar estos apuntes: usa el índice lateral para saltar de sección en sección. Cada bloque resume la información que normalmente aparece en una chuleta de HTML5/CSS3: etiquetas, atributos, propiedades, valores y ejemplos de uso.

HTML5

HTML5 define la estructura de una página web usando elementos semánticos, atributos y una jerarquía clara del documento.

Estructura básica del documento

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <title>Título de la página</title>
  </head>
  <body>
    Contenido...
  </body>
</html>
Elemento Descripción Notas
<!DOCTYPE html> Indica que el documento es HTML5. Debe ir al inicio del archivo.
<html> Elemento raíz del documento. El atributo lang indica el idioma principal.
<head> Contiene metadatos, título, enlaces a CSS, etc. No se muestra directamente en la página.
<body> Contenido visible de la página. Todo lo que el usuario ve va aquí.

Etiquetas semánticas de estructura

Etiqueta Uso principal
<header> Cabecera de la página o de una sección.
<nav> Zona de navegación (menús, enlaces principales).
<main> Contenido principal y único del documento.
<section> Sección genérica con contenido relacionado.
<article> Contenido independiente (post, noticia, etc.).
<aside> Contenido tangencial (barras laterales, notas).
<footer> Pie de página o de una sección.
Etiqueta Descripción
<div> Contenedor genérico sin significado semántico.
<span> Contenedor en línea genérico, para pequeños fragmentos.
<time> Representa una fecha u hora (legible para máquinas).
<figure> Ilustración, foto, código, etc. con <figcaption>.
<figcaption> Título o descripción de <figure>.

Texto, enlaces e imágenes

Elemento Ejemplo Uso
Títulos <h1>...</h1> hasta <h6> Estructuran el contenido jerárquicamente.
Párrafos <p>Texto...</p> Bloques de texto.
Enlaces <a href="https://...">Texto</a> Enlaces a otras páginas o secciones.
Imágenes <img src="img.png" alt="Descripción"> Mostrar imágenes. El atributo alt es obligatorio.
Énfasis <strong>, <em>, <mark> Marcan importancia, énfasis o resaltado.
Código <code>let x = 1;</code> Fragmentos de código en línea.

Listas y tablas

Listas
  • <ul>: lista no ordenada (viñetas).
  • <ol>: lista ordenada (números).
  • <li>: elemento de lista.
<ul>
  <li>Elemento 1</li>
  <li>Elemento 2</li>
</ul>
Tablas
  • <table>, <tr>, <td>, <th>
  • <thead>, <tbody>, <tfoot> para secciones. 
<table>
  <thead>
    <tr><th>Nombre</th><th>Edad</th></tr>
  </thead>
  <tbody>
    <tr><td>Ana</td><td>30</td></tr>
  </tbody>
</table>

Multimedia (audio y vídeo)

Elemento Ejemplo Atributos clave
<audio> <audio controls src="audio.mp3"></audio> controls, autoplay, loop, muted
<video> <video controls width="640" src="video.mp4"></video> controls, poster, width/height
<source> Formatos alternativos dentro de <audio>/<video>. src, type

Formularios HTML5

HTML5 introduce nuevos tipos de campos y atributos para validación.

Elemento / atributo Descripción Ejemplo
<form> Contenedor del formulario. <form action="/enviar" method="post">
<input type="email"> Campo de email con validación básica. <input type="email" required>
<input type="number"> Números con min, max, step. <input type="number" min="0" max="10">
<input type="date"> Selector de fecha. <input type="date">
required, pattern, placeholder Atributos de validación y ayuda visual. <input required pattern="\\d{5}" placeholder="C.P.">

Metadatos básicos (head / SEO)

<head>
  <meta charset="UTF-8">
  <title>Título descriptivo</title>
  <meta name="description" content="Descripción de la página">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link rel="stylesheet" href="styles.css">
</head>

CSS3

CSS3 se utiliza para definir la presentación de los documentos HTML: colores, tipografías, distribuciones de cajas, animaciones, etc.

Selectores básicos

Selector Ejemplo Descripción
Elemento p { ... } Selecciona todas las etiquetas <p>.
Clase .boton { ... } Elementos con class="boton".
ID #principal { ... } Elemento con id="principal".
Descendiente nav a { ... } <a> dentro de <nav>.
Hijo directo ul > li { ... } Solo hijos directos.
Atributo input[type="email"] Elementos con atributo/valor concreto.

Pseudo-clases y pseudo-elementos

Pseudo-clases (estado)
Selector Uso
a:link Enlace no visitado.
a:visited Enlace visitado.
a:hover Cuando el ratón pasa por encima.
a:active Durante el clic.
input:focus Campo que tiene el foco.
input:disabled, :enabled Campos desactivados/activados.
input:checked Checkbox o radio seleccionado.
:first-child, :last-child Primer/último hijo de un contenedor.
Pseudo-elementos (partes)
Selector Descripción
p::first-line Primera línea del párrafo.
p::first-letter Primera letra (capital inicial).
::before, ::after Contenido generado antes/después del elemento.
::selection Parte del texto seleccionada por el usuario.

Box model, margen, padding y posicionamiento

Box model
  • width / height: ancho y alto.
  • margin: espacio externo.
  • padding: espacio interno.
  • border: borde de la caja.
  • box-sizing: border-box; incluye el borde y padding en el ancho total. 
.caja {
  width: 300px;
  padding: 1rem;
  margin: 1rem auto;
  border: 1px solid #1f2937;
  box-sizing: border-box;
}
Posicionamiento
Valor Comportamiento
static Posición por defecto.
relative Desplazado respecto a su posición normal.
absolute Relativo al contenedor posicionado más cercano.
fixed Fijo respecto a la ventana (viewport).
sticky Se comporta como relativo y se “pega” al hacer scroll.

Flexbox

Flexbox sirve para organizar elementos en fila o columna de forma flexible. 

.contenedor {
  display: flex;
  flex-direction: row;      /* row | row-reverse | column | column-reverse */
  flex-wrap: nowrap;        /* nowrap | wrap | wrap-reverse */
  justify-content: center;  /* eje principal */
  align-items: center;      /* eje secundario */
}

.item {
  flex: 1 1 auto;           /* grow shrink basis */
  order: 0;                 /* orden visual */
}
Propiedad En contenedor En ítems
display display: flex; —
flex-direction row, column, etc. —
justify-content Alinea en el eje principal. —
align-items Alinea en el eje cruzado. —
flex — flex: grow shrink basis;
order — Cambia el orden visual de los ítems.

CSS Grid (introducción rápida)

.grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: auto;
  gap: 1rem;
}

.item-a {
  grid-column: 1 / span 2;
}
Propiedad Descripción
display: grid; Activa un contenedor de rejilla bidimensional.
grid-template-columns Define las columnas (número y tamaño).
grid-template-rows Define las filas.
gap Espacio entre filas y columnas.
grid-column, grid-row Posición de cada ítem en la rejilla.

Transiciones, transformaciones y opacidad

Transiciones 
.boton {
  transition: background-color 0.2s ease, transform 0.1s ease;
}

.boton:hover {
  background-color: #38bdf8;
  transform: translateY(-1px); 
}
Transformaciones / opacidad
  • transform: translateX(10px);
  • transform: scale(1.1);
  • transform: rotate(5deg);
  • opacity: 0.5; (0 = transparente, 1 = opaco)
